Welcome to genCounter!

genCounter is a webapp developed by Damon C. Roberts that allows users to upload a simple txt file to be analyzed with the Gendered Language Dictionary developed by Roberts and Utych (2019).

Use this webapp at your own risk. Do not upload files with sensitive information. This is a side project I developed while working on my dissertation. It is open source and you can access the code on my Github.

This webapp is free to use. But not free for me to build, run, and maintain. Please cite the use of this tool.

This project is currently under development!
